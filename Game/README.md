This is a game for development subject. 
The game was made by Víctor Tirado and Manel Mourelo.


Github repository: https://github.com/VictorTirado/Development-Game

In Ninja way you should get arrive to the final overcoming several obstacles.
Good luck!

Controls:
SPACE-> Jump
A-> Move to left
D-> Move to right
W->Climb

F1->? ?Start from the very first level
F2?-> ?Start from the beginning of the current level
F5?-> ?Save the current state
F6?-> ?Load the previous state
